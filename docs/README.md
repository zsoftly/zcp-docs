# Contributor Docs

Guides for writing and contributing to the ZSoftly documentation. Reader-facing docs live in
`src/content/docs/`.

For reporting issues and the fork/PR basics, see [CONTRIBUTING.md](../CONTRIBUTING.md). For
AI-assistant rules, see [CLAUDE.md](../CLAUDE.md).

| Doc                                            | Covers                                                 |
| ---------------------------------------------- | ------------------------------------------------------ |
| [content-standards.md](./content-standards.md) | Writing style, Starlight authoring rules, frontmatter  |
| [release-process.md](./release-process.md)     | Branch naming, commit format, local validation, the PR |
| [launch-checklist.md](./launch-checklist.md)   | Content and quality checks to run before publishing    |

Deployment (dev/stg/prd) is documented centrally in the private `zsoftly/platform` repo under
`docs/deployments/`, not here.
