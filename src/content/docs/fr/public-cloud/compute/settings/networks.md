## Configuration réseau

Gérez les interfaces réseau de la VM, y compris les options privées et publiques. Les réseaux privés
permettent la communication entre VM sans exposition à Internet. Les réseaux publics fournissent des
adresses IP externes pour l'accès Internet.

- Allez à **VM Settings** → **Réseaux**.
- Cliquez sur **Ajouter Réseau** pour ajouter une interface réseau.
- Cliquez sur un réseau pour modifier sa configuration, comme les limites de bande passante ou
  l'assignation IP.
