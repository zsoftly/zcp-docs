## Modifier le script de démarrage

Les scripts de démarrage s'exécutent automatiquement lorsque la VM est initialisée ou redémarrée.
Ils servent souvent à installer des logiciels, appliquer des mises à jour système ou configurer des
paramètres propres à une application.

- Allez à **VM Settings** → **Change Startup Script**.
- Ajoutez le script voulu, puis cliquez sur **Change**.

Vérifiez toujours la syntaxe du script avant de l'appliquer afin d'éviter les erreurs au démarrage.
