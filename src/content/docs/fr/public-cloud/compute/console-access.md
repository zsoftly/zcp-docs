La fonction **Accès à la console** vous permet de gérer votre VM et d'interagir avec elle au moyen
d'une interface Web. Elle est utile pour diagnostiquer les problèmes de démarrage, modifier la
configuration système ou interagir avec le système d'exploitation sans SSH ni RDP.

Accédez à la console en cliquant sur l'icône de console dans le menu d'actions.

## Raccourcis clavier

Les raccourcis suivants sont disponibles dans l'interface de console :

- **Toggle Control (Ctrl)** : simule la touche Control.
- **Toggle Alt** : simule la touche Alt.
- **Toggle Shift** : simule la touche Shift.
- **Toggle Windows Key** : simule la touche Windows.
- **Send Tab** : envoie une entrée Tab.
- **Send Escape (Esc)** : envoie une entrée Escape.
- **Send Ctrl+Alt+Delete** : envoie la combinaison Ctrl+Alt+Delete.
- **Clipboard** : facilite la copie de texte entre la machine locale et la VM.
- **Full-Screen Mode** : affiche la console de la VM en plein écran.

## Utilisations possibles

L'accès à la console vous permet de :

- installer des logiciels;
- ajuster les paramètres de configuration;
- diagnostiquer le système d'exploitation.
