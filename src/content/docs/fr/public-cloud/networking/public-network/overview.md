## Vue d'ensemble du réseau public

Un réseau public est un réseau défini logiquement dans l'environnement cloud. L'onglet **Overview**
affiche :

- **Réseau ID** : l'ID attribué au réseau.
- **Réseau Type** : Isolated (aucun accès externe) ou Cached (optimisation de stockage temporaire).
- **Traffic Types** : trafic Guest (orienté utilisateur) ou Public (accès Internet).
- **Gateway** : point d'accès entre les réseaux.
- **Netmask** : définit la plage IP du sous-réseau.
- **CIDR** : précise la plage IP du réseau.
- **Account** : compte propriétaire du réseau.
