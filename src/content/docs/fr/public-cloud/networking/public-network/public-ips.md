---
title: Adresses IP publiques
sidebar_position: 3
---

Les adresses IP publiques permettent aux ressources du réseau de communiquer avec Internet. Toutes
les IP publiques incluent une atténuation DDoS au niveau réseau, qui absorbe les attaques
volumétriques avant qu'elles n'atteignent vos charges de travail.

- Dans l'onglet **Adresses IP publiques**, consultez toutes les IP publiques assignées.
- Cliquez sur **Acquire New IP** pour demander une nouvelle IP publique.
- Choisissez le **cycle de facturation** voulu et cliquez sur **Buy IP**.
