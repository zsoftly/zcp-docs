## Liste des ACL réseau

Une **ACL réseau** agit comme un pare-feu qui contrôle le trafic entrant et sortant des sous-réseaux
dans le VPC.

- L'onglet **Réseau ACL List** affiche toutes les ACL configurées dans le VPC.
- Cliquez sur **Add New List** pour créer une nouvelle ACL.
- Fournissez un **Nom** et une **Description**.
- Cliquez sur **Soumettre**.
