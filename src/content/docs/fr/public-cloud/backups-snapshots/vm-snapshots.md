Les instantanés d'instance capturent l'état actuel d'une VM, y compris sa configuration et ses
données, à un moment précis. Utilisez-les pour la sauvegarde, la reprise après sinistre et les
tests.

### Créer un instantané de VM

- Dans le menu de gauche, cliquez sur **Instantanés** → onglet **Instances Snapshot**.
- Cliquez sur **Take Snapshot** ou sur l'icône **+**.

:::note

Captures d'écran à venir.

:::

### Étapes

1. **Emplacement** : sélectionnez le centre de données.
2. **Projet** : assignez l'instantané à un projet.
3. **Instance** : sélectionnez la VM à instantané.
4. **Nom de l'instantané** : fournissez un nom unique.
5. **Créer** : facturation : Hourly seulement, règle Fixed Prorata. Cliquez sur **Take Snapshot**.

:::note

Captures d'écran à venir.

:::
