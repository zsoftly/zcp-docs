La référence API interactive complète est hébergée dans Swagger UI :

- **API Cloud Platform** : [Ouvrir Swagger UI](https://api.zcp.zsoftly.ca/api/docs/nimbo)
- **API Stockage objet** : [Ouvrir Swagger UI](https://api.zcp.zsoftly.ca/api/docs/ceph)

Swagger UI fournit :

- La liste complète des points de terminaison avec les schémas de requête et de réponse
- Des tests de requêtes dans le navigateur (autorisez avec votre jeton Bearer)
- Le téléchargement de la spécification OpenAPI

### Domaines de l'API

| Domaine               | Description                                      |
| --------------------- | ------------------------------------------------ |
| Authentification      | Connexion, 2FA, réinitialisation du mot de passe |
| Machines virtuelles   | Gestion du cycle de vie des VM                   |
| Réseaux               | Réseaux publics et VPC                           |
| Stockage              | Volumes, instantanés, stockage objet             |
| Kubernetes            | Gestion des clusters                             |
| Équilibreur de charge | Répartition du trafic                            |
| DNS                   | Gestion des domaines et des enregistrements      |
| Facturation           | Factures, paiements et utilisation               |
| Soutien               | Billets et FAQ                                   |
