# Après le déploiement

Ce que ZSoftly livre et les premières étapes à suivre après la remise.

## Ce que vous recevez

Lors de la remise, ZSoftly fournit un document d'identifiants contenant :

- L'URL de l'interface de gestion CloudStack.
- Le nom d'utilisateur et le mot de passe administrateur CloudStack.
- Le fichier de configuration VPN pour l'accès à distance.
- L'URL et les identifiants du tableau de bord Ceph, le cas échéant.
- Les coordonnées du soutien ZSoftly.

## Liste de vérification initiale

- [ ] Connectez-vous à votre nuage privé au moyen du VPN : voir
      [Accès VPN](/fr/private-cloud/getting-started/vpn-access)
- [ ] Connectez-vous à l'interface de gestion CloudStack : voir
      [Accéder à CloudStack](/fr/private-cloud/getting-started/accessing-cloudstack)
- [ ] Changez le mot de passe administrateur par défaut
- [ ] Passez en revue la disposition de vos zones et clusters : voir
      [Configuration de zone](/fr/private-cloud/getting-started/zone-setup)
- [ ] Créez votre premier compte utilisateur dans CloudStack
- [ ] Testez la création d'une VM avec une image Linux de base

## Soutien

Si un élément ne fonctionne pas comme prévu au moment de la remise, communiquez immédiatement avec
le soutien ZSoftly. Le déploiement initial est couvert par l'entente de déploiement. N'attendez pas.
