# Soutien pour Private Cloud

## Contacter le soutien ZSoftly

Pour le soutien lié au nuage privé, utilisez les coordonnées fournies dans votre document de remise.

**Portail de soutien :** https://cloud.zcp.zsoftly.ca (ouvrez un billet sous votre compte de nuage
privé)

## Ce que ZSoftly couvre

- Les problèmes de déploiement initial, c'est-à-dire tout élément qui ne fonctionne pas à la remise.
- Les problèmes de couche infrastructure, comme les pannes d'hôtes, les problèmes de stockage et les
  interruptions réseau.
- Les mises à niveau et correctifs CloudStack, selon ce qui est prévu à votre contrat.
- Les problèmes d'accès VPN.
- L'expansion de capacité, comme l'ajout d'hôtes ou de stockage.

## Ce que vous gérez

- Le provisionnement et la gestion des VM.
- La configuration applicative dans vos VM.
- La gestion des utilisateurs et des comptes dans CloudStack.
- La gestion des compartiments et des objets dans Ceph RGW.
- Les charges de travail Kubernetes, si elles sont déployées.

## Avant de contacter le soutien

1. Vérifiez les alertes ou messages d'erreur dans l'interface CloudStack.
2. Consultez `sudo wg show` si le problème est lié au VPN.
3. Vérifiez l'état de santé dans le tableau de bord Ceph si le problème est lié au stockage.
4. Ayez en main votre **nom de zone** et votre **version de CloudStack**.
