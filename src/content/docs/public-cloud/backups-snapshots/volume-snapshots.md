## Volume Snapshots

Volume Snapshots capture the current state of a block storage volume. Use them to recover from
accidental deletion, corruption, or system failures.

### Create a Volume Snapshot

- From the left-hand menu, click **Snapshots** → **Volume Snapshot** tab.
- Click **Take Snapshot** or the **+** icon.

### Steps

1. **Location**: select the data center.
2. **Project**: assign to a project.
3. **Block Storage**: select the volume to snapshot.
4. **Snapshot Name**: provide a unique name.
5. **Create**: Billing: Hourly only, Fixed Prorata rule. Click **Take Snapshot**.

See also: [Create Volume](../storage/block-storage/create-volume), [VM Snapshots](./vm-snapshots)
