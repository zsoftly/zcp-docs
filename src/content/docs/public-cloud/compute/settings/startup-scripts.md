## Change Startup Script

Startup scripts execute automatically when the VM is initialized or rebooted. Common use cases
include installing software, applying system updates, or setting up application-specific
configurations.

- Go to **VM Settings** → **Change Startup Script**.
- Add your desired script and click **Change**.

Always review your script syntax before applying to avoid boot-time errors.
