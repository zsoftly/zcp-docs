The VM Instance Activity Log provides a detailed record of all actions and events related to a VM
instance, useful for monitoring and performance analysis.

- To view activity logs, go to the **Activity Logs** tab on the instance page.
- Track when a VM was created, stopped, deleted, rebooted, and more.
